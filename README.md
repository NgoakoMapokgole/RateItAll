RateItAll — User-Driven Rating Platform

RateItAll is a user-driven rating platform where anyone can share opinions and experiences about anything — from movies and restaurants to gadgets, books, and everyday topics.

Users can register, upload profile pictures, and post personalized reviews with images. The platform encourages open interaction through likes, dislikes, and messaging, creating a social space built on shared opinions. Administrators maintain fairness by managing users, removing inappropriate content, and keeping the community safe.

1. Motivation & Purpose

RateItAll was created to give users a genuine voice in an online space often dominated by biased or commercial reviews. The project focuses on authenticity, communication, and responsible community engagement.

Its purpose is to combine transparency and creativity while ensuring a safe environment that balances open expression with moderation.

2. Key Features

User Features

Create and edit profiles with bios and images

Follow other users and view their latest posts

Like or dislike ratings and comments

Report inappropriate content

View an activity feed of followed users

Content & Ratings

Upload images, videos, and audio with reviews

Categorize and tag posts for easy discovery

Sort and filter ratings by popularity, newest, or category

Admin Features

Manage users, posts, and reports via an admin dashboard

Assign user roles (admin, moderator, user)

Review and act on flagged content

View analytics and activity logs

System Features

Notifications for likes, follows, and messages

Responsive design for mobile and desktop

Security through password hashing and input validation

3. Technology Stack

Frontend: HTML5, CSS3, JavaScript, Font Awesome
Backend: PHP (server logic, authentication, sessions)
Database: MySQL
Libraries: Chart.js / Google Charts for analytics
Environment: CS3 Windows Server with local media storage

4. Educational Purpose

Developed as a school project to demonstrate full-stack web development skills, including database design, user authentication, media handling, and ethical data management.
